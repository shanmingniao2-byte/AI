概览

GLM多语言互译工具是一个集成了智谱AI大模型的多功能工具，不仅支持高质量的多语言互译，还专为AI文生图（Text-to-Image）用户设计，提供提示词扩写、反向提示词生成以及图片反推提示词等功能。

文本翻译

1. 输入文本：在左侧文本框中输入您要翻译的文本。

2. 选择语言：上方的下拉菜单用于选择源语言和目标语言。源语言支持“自动检测”。

3. 选择模型：根据需求选择合适的GLM模型（如GLM-4-Flash、GLM-4.5等）。

4. 执行翻译：点击“翻译”按钮或使用快捷键 Ctrl+Enter 开始翻译。结果将显示在右侧输出框中。

5. 复制结果：点击“复制翻译”按钮可将结果复制到剪贴板。

AI扩写

此功能可将简短的AI文生图提示词扩写成更详细、更专业的描述，从而生成更高质量的图像。

1. 在左侧文本框中输入您的基础提示词。

2. 点击底部的“扩写”按钮。

3. AI将为您生成一个更丰富的提示词。此时，“扩写”按钮会变为“原文”按钮。

4. 您可以随时点击“原文”按钮切换回原始提示词，再次点击可切换回扩写后的版本。

5. 扩写成功后，系统会自动生成一组“反向提示词”供您参考。

提示词库

内置了丰富的AI文生图提示词库，涵盖艺术风格、光照、构图、色彩等多个维度。

1. 点击底部的“提示词库”按钮打开库面板。

2. 通过顶部的标签页切换不同类别的提示词。

3. 添加提示词：点击任意一个提示词卡片，它会被自动添加到左侧的翻译/提示词输入框中，并用逗号分隔。

4. 移除提示词：再次点击已选中的（高亮的）提示词卡片，即可从输入框中移除它。

5. 提示词卡片支持拖拽，您可以将其拖拽到输入框、输出框或反向提示词框中。

反向提示词

反向提示词（Negative Prompt）用于告诉AI在生成图像时应避免哪些内容，是提升出图质量的关键。

1. 当您使用"AI扩写"功能后，系统会根据"自动生成"复选框的状态决定是否在下方生成一组反向提示词。

2. 如果您不希望自动生成反向提示词，可以取消勾选反向提示词区域右上角的"自动生成"复选框。

3. 您可以手动编辑此文本框，添加或删除内容。

4. 使用"复制"按钮可将其复制到剪贴板，以便在其他AI绘图工具中使用。

5. 您也可以将提示词库中的提示词拖拽到此文本框中，作为反向提示词的一部分。

图片反推提示词

此功能可以分析您上传的图片，并生成一段详细的、可用于AI文生图的提示词。

1. 在页面下方的“图片反推提示词”区域，点击上传区域或拖拽图片到此处。

2. 支持 JPG、PNG 格式，文件大小不能超过5MB。

3. 上传成功后，选择您希望生成的提示词语言（如中文、英文等）。

4. 点击“分析图片”按钮，AI将开始分析图片并生成提示词。

5. 生成后，您可以使用“复制”、“扩写提示词”或“使用此提示词”按钮进行后续操作。“使用此提示词”会将生成的内容直接填充到上方的主输入框中。

文生图功能

此功能可以根据您输入的提示词生成相应的图像，支持多种分辨率选择。

1. 在页面下方的"文生图功能"区域，输入正向提示词（推荐使用英文）。

2. 如有需要，可以在反向提示词框中输入您不希望出现在图像中的内容。

3. 从下拉菜单中选择您想要的图像分辨率（如1024×1024、1024×768等）。

4. 点击"生成图像"按钮，AI将使用CogView-4模型根据您的提示词生成图像。

5. 生成完成后，您可以使用"下载图像"按钮保存生成的图像。

重要提醒：文生图功能使用智谱AI的CogView-4模型，支持生成汉字并在语义理解、图像生成质量等方面有全面提升。
